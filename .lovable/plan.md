# خطة تحسين التصميم — Micro UI v2

بعد مراجعة 12 شاشة من الواجهة الحقيقية (الرئيسية، المصاريف، التحليلات، التذكيرات، البحث، الإعدادات وكل فروعها) على عرض هاتف فعلي (390×844px)، ظهرت هذه الملاحظات الموحّدة:

## نتائج المراجعة

**المشاكل الرئيسية:**
1. **الترويسة (Header)** ضخمة — ارتفاع ~64px وشعار 28px، تستهلك 8% من الشاشة بلا فائدة.
2. **بطاقة "إجمالي الأرصدة"** بارتفاع ~210px مع أرقام ضخمة (32px) وحشو 24px — تأكل نصف الشاشة قبل أي محتوى.
3. **بطاقات "الأرصدة حسب العملة"** مكررة عمودياً بارتفاع 130px لكل واحدة، تدفع قائمة الأشخاص خارج الشاشة.
4. **صفوف الإعدادات** بارتفاع 76px، أيقونات 40px، وعناوين 16px Bold — تبدو كقائمة iOS قديمة بدل لوحة تحكم محاسبية.
5. **بطاقات KPI في التحليلات** فارغة الوسط، حشو 24px، ارتفاع 110px لرقم واحد.
6. **شريط التنقل السفلي** بارتفاع 72px مع أيقونات 24px ونصوص 12px — يضغط على المحتوى.
7. **FAB + زر الذكاء العائم** كلاهما 56px ومتراكبان — يحجبان آخر بطاقة.
8. **عدم وجود سلّم مقاسات موحّد** — كل مكوّن يكرر قيم padding/font-size الخاصة به.

---

## التغييرات (4 مراحل قصيرة)

### المرحلة 1 — تأسيس سلّم المقاسات (Design tokens)
- في `src/styles.css` أضف رموز موحّدة:
  - مقاييس النص: `--text-2xs:10px / --text-xs:11 / --text-sm:12 / --text-base:13 / --text-md:14 / --text-lg:16 / --text-xl:18`.
  - مقاييس الحشو: `--space-1:4 / 2:6 / 3:8 / 4:10 / 5:12 / 6:14 / 7:16`.
  - مقاييس الأيقونات: `--icon-xs:12 / sm:14 / md:16 / lg:20`.
  - زوايا أصغر: `--radius-sm:6 / md:8 / lg:10` بدل 12-16 الحالي.
- خفض الأساس `html { font-size: 13px }` (حالياً 14) لينعكس على كل قياسات rem تلقائياً.

### المرحلة 2 — تكثيف Layout الرئيسي
- **Header** (`__root` / AppHeader): ارتفاع 44px، شعار 14px Bold، أيقونات 16px، حشو أفقي 10px.
- **Bottom Nav** (`BottomNav.tsx`): ارتفاع 52px، أيقونة 18px، نص 9.5px، gap 2px.
- **FAB + Smart Add**: تصغير لـ 44px وتجميعهما عمودياً بمسافة 8px.

### المرحلة 3 — البطاقات (الجزء الأهم)
- **TotalsCard** (الديون): ارتفاع 100px بدل 210، رقم رئيسي 22px، حواف 8px، عمودان جنباً لجنب بحشو 8px.
- **MultiCurrencyTotals**: تحويل كل عملة إلى صف مدمج (chip) بارتفاع 56px بدل بطاقة 130px — رمز العملة + لك/عليك أفقياً.
- **PersonRow / TransactionRow**: ارتفاع 48px، Avatar 30px، عنوان 12.5px، وصف 10.5px.
- **KpiCard** (التحليلات): شبكة 3 أعمدة بدل 2، ارتفاع 72px لكل بطاقة، رقم 18px.
- **SettingsRow**: ارتفاع 44px، IconTile 26px، عنوان 12.5px، وصف 10.5px، ChevronLeft 12px.
- **SettingsGroup**: حشو داخلي 4px، فاصل صف 1px شفاف.
- **بطاقة الملف الشخصي** أعلى الإعدادات: ارتفاع 48px بدل 72px.

### المرحلة 4 — تطبيق وفحص بصري
- تحديث `src/components/common/{IconTile, SettingsRow, SettingsGroup, FabButton, Avatar}.tsx` و`BottomNav.tsx` و`features/debts/*` و`features/insights/KpiCard.tsx` لاستهلاك التوكنز الجديدة.
- إعادة التقاط نفس الـ 12 شاشة بـ Playwright والمقارنة قبل/بعد.
- التأكد من توافق الوضع الداكن (لا تغيير ألوان، فقط مقاسات).

## ما لن يتغير
- الألوان الدلالية والهوية (أزرق/أخضر/أحمر) — التركيز فقط على المقاسات والكثافة.
- البنية المنطقية والمسارات — تغييرات عرض فقط.
- لا تعديل في الواجهة الخلفية ولا منطق الأعمال.
